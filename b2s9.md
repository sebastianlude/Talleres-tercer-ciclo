### Problema 1
**Sincronizaci√≥n de Sensores**

Descripci√≥n: Simula dos sensores que emiten valores peri√≥dicos:

- Sensor1: Rango [20¬∞C, 30¬∞C], cada 1 segundo.
- Sensor2: Rango [30¬∞C, 40¬∞C], cada 1.5 segundos.
- Usa zip para combinar los valores de ambos sensores en un solo flujo.
- Det√©n el flujo despu√©s de 10 pares de valores.
- Genera una alerta si la diferencia absoluta entre las lecturas de Sensor1 y Sensor2 es mayor a 5¬∞C.
  

```mathematica
package Clases.Semana9

import rx.lang.scala.Observable
import scala.util.Random
import scala.concurrent.duration._

object SincronizacionSensores extends App {
  val sensor1 = Observable
    .interval(1.second).map(_ => Random.between(20.0, 30.0))

  val sensor2 = Observable
    .interval(1.5.second).map(_ => Random.between(30.0, 40.0))

  val combinacion = sensor1.zip(sensor2)
  combinacion
    .take(10)
    .doOnNext { case (s1, s2) =>
      println(f"Sensor1: $s1%.2f¬∞C, Sensor2: $s2%.2f¬∞C, Diferencia: ${math.abs(s1 - s2)}%.2f¬∞C")
    }
    .filter { case (s1, s2) =>
      math.abs(s1 - s2) > 5
    }
    .map { case (s1, s2) =>
      val diferencia = math.abs(s1 - s2)
      f" ALERT! Significant difference detected: $diferencia%.2f¬∞C"
    }
    .subscribe(
      msg => println(msg),
      e => println("Error: " + e.getMessage),
      () => println("Fin")
    )
  Thread.sleep(20000)
}




```
### Problema 2
**Sistema de Alerta Complejo**

Descripci√≥n: Simula tres sensores que emiten datos peri√≥dicos:

- Sensor1: Temperatura (20¬∞C a 30¬∞C).
- Sensor2: Humedad (40% a 60%).
- Sensor3: Presi√≥n (950 hPa a 1050 hPa).
- Combina los datos de los tres sensores y detecta si alguna de las m√©tricas est√° fuera de rango.
- Det√©n el flujo despu√©s de 20 combinaciones.

```yaml
package Clases.Semana9

import rx.lang.scala.Observable
import scala.concurrent.duration._
import scala.util.Random

object SistemaAlertaComplejo extends App {

  val temperatura = Observable.interval(1.second)
    .map(_ => Random.between(20.0, 30.0))
  val humedad = Observable.interval(1.2.second)
    .map(_ => Random.between(40.0, 60.0))
  val presion = Observable.interval(0.9.second)
    .map(_ => Random.between(950.0, 1050.0))

  val combinados = Observable.zip(temperatura, humedad, presion)

  combinados
    .take(20)
    .doOnNext { case (t, h, p) =>
      println(f"Temp: $t%.2f¬∞C, Hum: $h%.2f%%, Pres: $p%.2f hPa")
    }
    .filter { case (t, h, p) =>
      t < 20 || t > 30 || h < 40 || h > 60 || p < 950 || p > 1050
    }
    .map { case (t, h, p) =>
      if (t < 20 || t > 30) f"üö® ALERT! Temperature out of range: $t%.2f¬∞C"
      else if (h < 40 || h > 60) f"üö® ALERT! Humidity out of range: $h%.2f%%"
      else f"üö® ALERT! Pressure out of range: $p%.2f hPa"
    }
    .subscribe(
      msg => println(msg),
      e => println("Error: " + e.getMessage),
      () => println("Monitoring stopped")
    )

  Thread.sleep(20000)
}



```
